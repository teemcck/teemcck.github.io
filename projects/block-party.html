Block Party

A rhythm game devlog about timing windows, chart readability, and presentation energy.

Itch GitHub

Overview

Block Party is a Friday Night Funkin'-inspired rhythm game focused on timing, readable chart feedback, and presentation that supports the beat.

01 / Prototype

Note interaction

The first pass made sure the chart/input loop worked before adding polish.

  • Chart-driven prompts
  • Input-to-note matching
  • FNF-style starting format

02 / Timing Pass

Timing windows

Misses need to feel understandable, not random.

  • Hit and miss windows
  • Immediate input response
  • Feedback before flourish

03 / Presentation

Readable energy

The presentation should support the beat without covering the chart.

  • Separate gameplay elements
  • Personality around the chart
  • Future pass: chart design notes
