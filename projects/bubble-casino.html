Bubble Casino

A blackjack game devlog focused on rules clarity, round flow, and comedic tone.

Itch GitHub

Overview

Bubble Casino is a blackjack simulator wrapped in a strange wedding-money premise. The devlog tracks the rules, round clarity, and presentation pass.

01 / Prototype

Card rules first

The first pass was mostly about getting the hand logic right.

  • Player and dealer turns
  • Hand value tracking
  • Win, loss, and bust states

02 / Table Flow

Readable rounds

The player should know the state of the hand before choosing an action.

  • Visible card state
  • Clear action timing
  • Round result feedback

03 / Polish Pass

Tone pass

The joke should sit around the blackjack, not bury it.

  • Wedding/casino framing
  • Stronger table feedback
  • Future pass: short retrospective
