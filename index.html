Game Jam Projects

Global Game Jam 2025

A 48-hour game jam submission inspired by the theme "Bubble." Set in a vividly colorful underwater casino, the game challenges players to win as much money as possible to fund their dream wedding. Featuring a fully functional blackjack game and an engaging storyline, this project was developed using C#11 and the Unity Game Engine.

MGDC Bluebird Studios Game Jam

48 hour game jam submission following the theme "Real world issues." This game is a roguelike shooter where the objective is to combat viruses infecting your computer. There are 7 different unlockable upgrades, 2 different bosses, and a developed story line. This project was developed using C#11 and the Unity Game Engine.

WolverineSoft 2024 Pumpkin Game Jam

48 hour game jam submission following the theme "Block Party." This game is a 2-player dynamic rhythm-based game where players synchronize actions to musical beats to outperform their opponent. There are 3 unique tracks, a regular and halloween theme, and 9 unique characters to select from. This project was designed using C#11 and the Unity Game Engine.

Independant Projects

Manada Game Development Engine

Manada engine is a simple, lightweight 3D engine designed for simulation, educational use, and game development. It was built with emphasis on clarity and modularity with support for basic rigid body physics. This project was designed using C++20, NVIDIA PhysX SDK, and the OpenGL API, with a focus on simplicity and ease of use.
